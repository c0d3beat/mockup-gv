Pembelian melalui web browser

Kunjungi halaman www.8ternals.com dan login menggunakan User ID dan Password Anda.


Pilih menu "TopUp" untuk melakukan pengisian Token.


Pada menu Metode Pembayaran pilih pembayaran "VOUCHER GAME".


Kemudian akan tampil pilihan pembayaran, pilih "GUDANG VOUCHER" untuk melanjutkan pembelian.


Kemudian pilih denominasi pengisian Token dan klik "Submit" untuk melanjutkan pembelian.


Halaman pembayaran dengan GudangVoucher akan tampil, kemudian Login menggunakan ID GudangVoucher Anda.


Pada halaman konfirmasi akan tampil tagihan yang akan mengurangi saldo Anda di GudangVoucher. Kemudian klik "Bayar" untuk melakukan pembayaran atas transaksi Anda.


Transaksi sukses! Kode yang tampil sebagai bukti transaksi. Klik "Back to Merchant".

Pembelian melalui mobile

Masuk ke dalam game aktif. Kemudian pilih "Beli" untuk melakukan pengisian Ball.


Pilih tombol "Google" untuk melakukan pengisian Ball dengan Google Wallet, atau pilih tombol “Token” untuk melakukan pengisian Ball dengan Token 8ternals.


Lakukan pembelian Ball dengan memilih denominasi yang Anda inginkan. Jika saldo Token Anda tidak cukup, Anda bisa memilih salah satu tombol "ISI TOKEN" yang tersedia.


Pilih "VOUCHER GAME" untuk melakukan pengisian Token.


Pilih "GUDANG VOUCHER" untuk melanjutkan pembelian Token Anda.


Kemudian pilih denominasi pengisian Token Anda dan klik tombol "OK" untuk melanjutkan pembelian.


Halaman pembayaran dengan GudangVoucher akan tampil, kemudian Login menggunakan ID GudangVoucher Anda.


Pada halaman konfirmasi akan tampil tagihan yang akan mengurangi saldo Anda di GudangVoucher. Kemudian klik "Bayar" untuk melakukan pembayaran atas transaksi Anda.


Transaksi sukses! Kode yang tampil sebagai bukti transaksi. Klik "Back to Merchant".
